**Amazon Clone Website 🛒**

Overview:
This project is a front-end clone of the Amazon homepage, built entirely using HTML5 and CSS3. It recreates the look and feel of one of the world's most popular e-commerce platforms while focusing on clean design, responsive layouts, and modern web development practices.
The project was created to strengthen my understanding of web design fundamentals, page structuring, styling techniques, and responsive user interface development.

✨ Features:
->Responsive Navigation Bar
->Amazon-inspired Homepage Design
->Hero Banner Section
->Product Category Cards
->Interactive Hover Effects
->Responsive Layout using Flexbox
->Organized and Readable Code Structure
->Footer Section Similar to Amazon
->Built Using Pure HTML5 and CSS3

🛠️ Technologies Used:
HTML5
CSS3
Flexbox
Responsive Web Design Principles

🎯 Learning Outcomes:
Through this project, I gained hands-on experience with:
->Semantic HTML Structure
->CSS Styling and Layout Techniques
->Flexbox Positioning
->Responsive Design Concepts
->UI Replication from Real-World Websites
->Project Organization and Best Practices
